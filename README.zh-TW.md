# ChatGPT 批次工具｜繁體中文使用說明

> 🇬🇧 English version: [README.md](./README.md)

這是一個 Chrome 瀏覽器的擴充功能（外掛），可以幫你**一次處理很多則 ChatGPT 對話**：
批次刪除、批次搬進專案、把過去的對話內容組合成新的素材、以及快速改名。

這份說明寫得比較詳細，**就算你完全沒寫過程式也能照著一步一步完成**。整個過程不用付費、不用註冊任何東西，只要你有一個能正常使用的 ChatGPT 帳號。

## 一、這個工具能做什麼

ChatGPT 官方網站上，要整理對話很麻煩——刪除、搬進專案、改名都得一則一則手動點。這個工具把這些變成「打勾 → 一次處理」。

它有四個主要功能：

- **🗑️ 批次刪除（Delete）**：勾選多則對話，一次刪掉。
- **📁 批次移至專案（Move to Project）**：把勾選的對話一次搬進某個專案，可以選現有的專案，也可以當場新建一個。
- **🧩 組裝素材（Build Context）**：這是最特別的功能。你常常會有兩三個相關的對話，各自有一些重要內容。這個功能讓你把這些對話裡「想要的段落」勾選出來，組合成一段文字，然後一鍵複製，或直接開一個新對話帶進去。適合用來「整理」「摘要」「把多個討論合併成大綱」。
- **✏️ 快速改名（Rename）**：滑鼠移到對話上，點鉛筆圖示就能直接改名字，不用像官方那樣點開好幾層。

## 二、安裝前的準備

你只需要兩個東西：

1. **Google Chrome 瀏覽器**（或 Edge 等同樣基於 Chromium 的瀏覽器）。
   - 還沒有的話，到 <https://www.google.com/chrome/> 下載安裝。
2. **一個 ChatGPT 帳號**，而且能在 <https://chatgpt.com> 正常登入使用。

> 💡 這個工具是「在你自己的電腦上、用你自己的帳號」操作，不會把你的資料傳給任何第三方。詳見最後的[隱私說明](#八隱私說明)。

## 三、下載這個工具（兩種方法）

擇一即可。**如果你不熟 Git，建議用「方法 A：直接下載 ZIP」最簡單。**

### 方法 A：直接下載 ZIP（推薦給非工程背景的朋友）

1. 打開這個專案的 GitHub 網頁。
2. 找到頁面右上方一顆綠色的按鈕 **`< > Code`**，點它。
3. 在跳出的小選單最下面，點 **`Download ZIP`**。
4. 瀏覽器會下載一個 `.zip` 壓縮檔（通常在「下載」資料夾）。
5. 找到這個檔案，**解壓縮**：
   - Windows：對著檔案按右鍵 →「全部解壓縮」。
   - Mac：直接點兩下檔案即可解開。
6. 解開後會得到一個資料夾（裡面有 `manifest.json`、`popup.html` 等檔案）。
   **記住這個資料夾放在哪裡**，等一下要用。

> ⚠️ 注意：請把解壓縮後的資料夾放在一個固定、不會被刪掉的位置（例如「文件」資料夾裡）。因為擴充功能安裝後會一直「讀取」這個資料夾，如果你把它刪掉或移走，擴充功能就會失效。

### 方法 B：用 Git 下載（給熟悉指令的人）

```bash
git clone <這個專案的網址>
cd gpt-tool-boxs
```

之後要更新時，在資料夾裡執行 `git pull` 就能拿到最新版。

## 四、安裝到 Chrome 瀏覽器

1. 打開 Chrome，在最上面的網址列輸入這串，然後按 Enter：

   ```
   chrome://extensions
   ```

2. 看到「擴充功能」管理頁面。把右上角的 **「開發人員模式 / Developer mode」** 開關打開。
3. 開關打開後，左上方會出現幾顆按鈕。點 **「載入未封裝項目 / Load unpacked」**。
4. 這時會跳出選擇資料夾的視窗。**選你剛剛在第三步解壓縮出來的那個資料夾**，然後按確定。
5. 成功的話，你會在頁面上看到「ChatGPT Batch Tools」這個擴充功能出現。
6. （建議）點 Chrome 右上角的拼圖圖示 🧩，找到這個工具，點旁邊的圖釘把它「釘選」起來，之後比較好找。

> 💡 安裝後，工具的圖示會出現在 Chrome 右上角。之後要用，就點那個圖示。

## 五、開始使用

1. 先打開 <https://chatgpt.com>，**確認你已經登入**。
2. 點 Chrome 右上角的工具圖示，會跳出一個小視窗（popup）。
3. 它會自動載入你最近的對話。如果想載入「全部」對話，點 **`Load All`**。
4. 視窗最上面有三個模式可以切換：**Delete（刪除）**、**Move to Project（移至專案）**、**Build Context（組裝素材）**。
5. 用每則對話左邊的方框打勾來選取，或點 **`Select All`** 全選。
6. 選好之後，點右上角那顆主要按鈕執行（按鈕文字會跟著你選的模式改變）。

## 六、四大功能詳細教學

### 🗑️ 批次刪除

1. 切到 **Delete** 模式。
2. 勾選你要刪除的對話。
3. 點 **`Delete Selected`**，會跳出確認視窗，再按一次確認才會真的刪除。

> ⚠️ **刪除無法復原**，按確認前請再看一次勾了哪些。

### 📁 批次移至專案

1. 切到 **Move to Project** 模式。
2. 勾選要搬動的對話。
3. 點 **`Move to Project`**，跳出專案選擇視窗：
   - 從現有專案清單挑一個，或
   - 點 **`+ New project`** 輸入名稱、當場新建一個專案。
4. 按確認，工具會一則一則把它們搬進去，並顯示進度。
5. 搬進專案的對話會從「最近對話」清單消失（因為它們現在屬於那個專案了）。

> 💡 搬動是可以反悔的——之後在 chatgpt.com 上可以把對話再移出專案。

### 🧩 組裝素材（Build Context）— 最推薦的功能

這個功能讓你把「散落在不同對話裡的重要片段」集中起來變成一段文字。

1. 切到 **Build Context** 模式，勾選你想取材的對話（可以選好幾個）。
2. 點 **`Build Context`**，跳出組裝視窗。
3. 視窗裡每則對話預設是**收合**的。點對話標題可以展開，看到裡面一段一段的問答。
4. 每段問答前面都有打勾框，**勾選你想帶走的段落**（預設全部勾選）。
   - 想看某段的完整內容？把滑鼠移上去，點右邊出現的 **⤢** 圖示，會彈出完整內容。
   - 想看整則對話完整內容？點該對話標題列右邊的 **`View`**。
   - 每則對話也有 **`All / None`** 可以快速全選/全不選那一則的段落。
5. 選好後，在下面挑選：
   - **格式（Format）**：Markdown（一般推薦）/ XML / Plain（純文字）。
   - **樣板（Template）**：要不要在素材後面自動加一句指示。
     - `None`：只給素材，不加指示。
     - `Summarize`：加上「請幫我摘要重點」。
     - `Outline`：加上「請幫我整理成大綱」。
     - `Reorganize`：加上「請幫我重新整理成清楚的內容」。
6. 最後選輸出方式：
   - **`Copy`**：複製到剪貼簿。之後你可以貼到任何地方——新的 ChatGPT 對話、筆記軟體都行。
   - **`Send to New Chat`**：直接幫你開一個新的 ChatGPT 分頁（在背景），並把組好的內容自動打進輸入框。切到那個分頁，按 Enter 就能送出（有些情況會自動送出）。

> 💡 為什麼不直接幫你送出並切過去？因為一旦切換視窗，這個小工具視窗就會自動關閉。所以新分頁會開在背景，由你自己切過去查看。

### ✏️ 快速改名

整理完之後，幫對話取個好名字很重要。

1. 在任何模式的對話清單，把滑鼠移到某則對話上。
2. 右邊會出現一個鉛筆圖示 **✎**，點它。
3. 標題會變成可編輯狀態，直接打新名字。
4. 按 **Enter** 儲存，按 **Esc** 取消。

## 七、常見問題

**Q：點開工具，卻顯示要我登入？**
A：請先到 <https://chatgpt.com> 確認登入成功，再重新點工具圖示。

**Q：「Send to New Chat」開了新分頁，但內容沒打進去？**
A：可能是 ChatGPT 改版導致輸入框位置變了。這種情況請改用 **`Copy`**，再自己貼上即可。

**Q：我把下載的資料夾刪掉/搬走了，工具壞了？**
A：擴充功能會持續讀取那個資料夾。請回到 `chrome://extensions` 把它移除，重新下載並放到固定位置後再重新「載入未封裝項目」。

**Q：更新到最新版怎麼做？**
A：重新下載一次（或用 `git pull`），然後到 `chrome://extensions`，在這個工具的卡片上點「重新載入 / Reload」。如果有新增權限，Chrome 可能會要你再確認一次。

**Q：刪錯對話能救回來嗎？**
A：從這個工具刪除無法復原，請務必在確認前看清楚勾了哪些。

## 八、隱私說明

- 這個工具**只跟 `chatgpt.com` 溝通**，不會連到任何其他伺服器。
- 它會讀取你在 ChatGPT 的登入憑證（session token），用來列出 / 刪除 / 搬動 / 改名你的對話、以及列出 / 新建專案。
- 這個憑證**只存在記憶體裡，用完即丟**，不會被存到硬碟、也不會傳給任何第三方。
- 所有操作都在你自己的電腦、用你自己的帳號進行。
